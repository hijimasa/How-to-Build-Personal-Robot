# 付録　クローンボードの取り扱い

!!! note "この章のステータス"
    ドラフト作成中。本文は未執筆です。執筆方針は [CONCEPT.md §13 執筆ルール](../CONCEPT.md) を参照。

本書は **公式品の特定型番** でのみ動作を保証します。クローン品・互換品を否定はしませんが、
本編で条件分岐を増やすと初心者が迷子になるため、クローン固有の話題は本付録に集約します。

## この付録の対象読者

- 公式品の入手が難しく、クローン品で代用したい読者
- クローン品で購入したあとに「本書の手順通りに動かない」と感じた読者
- 公式品とクローン品の違いを、仕組みから理解したい読者

## 目次（予定）

1. クローン品とは何か（互換品／偽造品／設計流用の違い）
2. USB シリアルドライバの差（CH340 / CP2102 / FT232RL）と導入手順
3. 電源レギュレータの品質による挙動差
4. 偽造チップによる書き込み不能症状の典型
5. クローン品を使うときの見極めチェックリスト
6. 「クローンで動かない」読者報告の受け口（本編ではなくここで受ける）
